# my-apps-showcase

Vitrine no GitHub dos projetos que desenvolvo com IA e linguagens de programação.

Cada pasta é um app independente, com seu próprio README, repositório e instruções de uso.

## Projetos

| Projeto | Descrição |
|---|---|
| [myapp-cache-cleaner](./myapp-cache-cleaner) | Extensão Chrome para limpar cache do navegador em qualquer site |

## English

**my-apps-showcase** is the parent repository for my GitHub portfolio. It groups small apps and tools I build with AI assistance and programming languages.

Each subfolder is a standalone project with its own README, repository, and setup instructions. Start from the project folder you need — the details live there, not in this file.
